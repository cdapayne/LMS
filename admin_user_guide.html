Dashboard

The dashboard offers a snapshot of current activity and links to common tasks.

  • Stats cards: show counts of classes, students, teachers and pending applications. Click a card to open the related management page.
  • Quick Actions: provide shortcuts to student queues, class tools, reports, events, marketing, email templates and more. Review these actions often to keep operations moving.
  • Announcements: post messages to students, teachers or everyone. Keep announcements brief and include clear calls to action.
  • Charts: visualize trends such as monthly sign‑ups. Select the edit icon to configure chart type, data source and date range.

Students

Pending Students

Review new registrations awaiting approval. Open a profile to verify documents and information. Approve to grant access or decline to deny enrollment. Use notes to record reasons for decisions.

Accepted Students

Lists all approved students. From a profile you can reset passwords, sign documents, send follow‑up emails or archive records when a student leaves.

Denied Students

Shows previously declined applications. Revisit and approve later if circumstances change.

Pre‑Registered Users

Displays users who expressed interest but have not completed registration. Follow up with reminders to encourage completion.

Teachers

Create new teacher accounts and manage existing staff.

  • Create Teacher: enter profile information, assign a temporary password and designate permissions.
  • Teacher List: browse all teachers, open profiles to reset credentials, view classes or disable access.

Classes

Admins control all classes offered in the system.

  • Class List: search, filter and export to CSV. Open a class for detailed management.
  • Create Class: define core details, assign a teacher, set schedule and duration. Double‑check dates before publishing.
  • Manage Class: within a class you may add students, upload lectures or PowerPoint files, create tests, attach media and host discussions. Use the Preview button to verify the student view.

Events

  • Events Dashboard: overview of upcoming events with links to edit or remove.
  • Create Event: publish events with date, description and optional attachment. Mark events as public or private.
  • Event RSVPs: track attendee responses and export the list for on‑site check‑in.

Marketing & Drip Campaigns

Marketing Emails

Send targeted messages to students. Choose recipients, pick a type (recruitment, retention, approval, events or information), craft a subject and message, preview using AI, then send. Monitor open rates to refine future campaigns.

Drip Campaigns

Schedule automated follow‑ups. Provide contact information and segment; active campaigns display their current step. Review campaign performance regularly and pause sequences that are no longer needed.

Email Templates

Manage reusable email templates. Select a template, edit subject and HTML body, preview the rendered email, or generate content with AI. Save to update the template library and reuse across campaigns.

Branding

Customize the logo and the primary and secondary colors used throughout the site. Upload a new logo image or choose colors and click Save to apply. A consistent brand builds trust with students and partners.

Dropdown Options

Add or remove values for course names and affiliate programs that appear in student forms. Keep lists current to avoid confusion during registration.

Reports

  • Class Summary: view counts of students, tests and grades for each class. Export the report to share with stakeholders.
  • Pending Students Report: analyze applications awaiting approval and identify bottlenecks in processing.
  • Custom Report Builder: select data fields to generate custom CSV exports. Save report templates for recurring use.

Announcements

Publish messages to all users, only teachers or only students. Use announcements to highlight deadlines, policy changes or achievements. Announcements appear on dashboards immediately after posting.

Charts

The dashboard contains configurable charts for tracking sign‑ups and other metrics. Use the gear icon to choose chart type, data table and fields. Share charts during planning meetings to support data‑driven decisions.

Support & Best Practices

Regularly review audit logs and user activity for security. Provide training for new staff and document internal procedures. For questions or assistance, contact the system administrator.

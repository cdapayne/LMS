Dashboard

The dashboard provides quick stats and links to common tasks.

  • Stats cards: show the number of classes, students, and teachers.
  • Quick Actions: shortcut buttons to student queues, class tools, reports, events, marketing, email templates and more.
  • Announcements: post a message to students, teachers or everyone.
  • Charts: visualize sign‑up trends. Use the edit icon to configure chart type and data source.

Students

Pending Students

View new registrations awaiting approval. Approve to grant access or decline to deny enrollment. Each student profile allows you to review submitted information and required documents.

Accepted Students

List of approved students. Open a profile to reset passwords, sign documents, or trigger step‑2 enrollment emails.

Denied Students

Review previously declined applications. You can revisit the decision if needed.

Pre‑Registered Users

Shows users that expressed interest but have not started the full registration.

Teachers

Create new teacher accounts and manage existing teachers.

  • Create Teacher: supply profile information and login credentials.
  • Teacher List: browse all teachers and open individual profiles.

Classes

Admins control all classes offered in the system.

  • Class List: search, export CSV, and open classes for detailed management.
  • Create Class: define core details, assign a teacher, set schedule and duration.
  • Manage Class: within a class you may add students, upload lectures or PowerPoint files, create tests, attach media, and host discussions.

Events

  • Events Dashboard: overview of upcoming events.
  • Create Event: publish events with date, description and optional attachment.
  • Event RSVPs: view attendee responses.

Marketing & Drip Campaigns

Marketing Emails

Send targeted messages to students. Choose recipients, pick a type (recruitment, retention, approval, events or information), craft a subject and message, preview using AI, then send.

Drip Campaigns

Schedule automated follow‑ups. Provide contact information and segment; active campaigns display their current step.

Email Templates

Manage reusable email templates. Select a template, edit subject and HTML body, preview the rendered email, or generate content with AI. Save to update the template library.

Branding

Customize the logo and the primary and secondary colors used throughout the site. Upload a new logo image or choose colors and click Save to apply.

Dropdown Options

Add or remove values for course names and affiliate programs that appear in student forms.

Reports

  • Class Summary: view counts of students, tests, and grades for each class.
  • Pending Students Report: analyze applications awaiting approval.
  • Custom Report Builder: select data fields to generate custom CSV exports.

Announcements

Publish messages to all users, only teachers, or only students. Announcements appear on dashboards after posting.

Charts

The dashboard contains configurable charts for tracking sign‑ups. Use the gear icon to choose chart type, data table and fields.

For questions or assistance, contact the system administrator.

Admin User Guide

This guide explains how to manage the learning management system as an administrator.

Branding

Navigate to Branding in the navigation bar to update the site logo and the primary and secondary colors.

  1. Choose a new logo file and/or pick your desired colors.
  2. Click Save to apply your changes across the application.

Other Tasks

  • Approve or deny student registrations.
  • Manage classes and events.
  • Create email templates and configure dropdown options.
